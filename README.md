# Online Personal Profile

React + Vite personal profile site for Paul Mbugua.

## Local Development

```powershell
npm run dev
```

## Cloudflare Pages

Use these settings when connecting this repository to Cloudflare Pages:

- Build command: `npm run build`
- Build output directory: `my Profile/dist`
- Root directory: repository root
- Node.js version: 20

The Vite app source lives in `my Profile/`. The root build script installs and builds that app for Cloudflare Pages.
